# Autoclicker

Autoclicker for Progress bar

How to Use:
This is for Cars Learning Driving School use

NOTE: The Script resets and will not work if you navigate to a different course unit, you will need to reopen the console again and then paste the script again and change/select the index.html file on the console, then press enter and it should work after seeing "Precision Clicker Activated"

1. Open the inspect part of the page and navigate to the console of the inspect section of the course unit you will want to complete
2. Go to the part where it says "top" on the console and click on it, then select the index.html file
3. Paste the script on the console and press enter
4. You should see "Precision Clicker Activated" on the console
5. After it reaches 100% of the progress bar it takes 2 seconds to click "next" and you can close the inspect part if you want but the page of the driving course must be open, so you might have to do split screen or open it at the same time
